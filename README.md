# Skillup
SkillUp is a **modern online learning platform** built with **React**. This project includes a homepage, dashboard, login, and signup forms with a clean UI.


## Features

- Fully responsive homepage with sections for:  
  - Hero  
  - About  
  - Courses  
  - Why Choose Us  
  - Contact Form  
  - Footer
- **Login** and **Signup** forms with validation  
- Dashboard to view courses, progress, and certificates  
- Smooth scrolling to sections  
- Clean CSS styling without frameworks  For now
